# UserManagementSystem

WebApp for CRUD operation using EclispeIDE

## Features

- Register / Login
- View all registered users 
- Delete user details
- Modify user details

## Languages used

- Java Servlet
- JSP
- Bootstap 4
- MYSQL
- JSTL

## HOW TO RUN
```
  Import repository from eclipse IDE
  Select tomcat as the targeted runtime
  Execute the project using tomcat server
```
